Clean Code

What is clean code?

Writing code can be a very messy process! The basic idea of clean code is to write something that can be read and understood by someone without the code creator’s help.
Whilst there are some ideas and styles that are considered ‘best practice’ within the coding community, it is often down to personal preference how code is structured. This page will explain what makes clean code clean, and give an idea of how you can make sure your own code is up to scratch!
